# KurzCademy
## Ako to rozbehať

1. Naclonovať repo
2. Rozbehať composer
```
composer update
```
3. Rozbehať october
```
php artisan october up
```
4. Buildnúť Vue aplikáciu
```
cd ./themes/vuetober
npm run build
```

## Spustenie pre vývojárov
1. Spustiť php
```
php artisan serve
```
2. Spustiť Vue
```
cd ./themes/vuetober
npm run serve
```
> Aj php aj Vue musia bežať naraz!
3. Otvoriť v prehliadači [localhost:8000](http://localhost:8000)
